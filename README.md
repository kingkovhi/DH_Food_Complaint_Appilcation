# DH_Food_Complaint_Appilcation

The DH_Food_Complaint_Appilcation   is an anonymous feedback platform where students can share concerns, complaints, or compliments about dining hall food and services.  
Users receive a unique ID to track whether their submission has been seen or reviewed (responded to) by staff.  
A community feed displays all anonymous posts ordered by date, showing response status so students can see progress.  
Staff have a secure admin dashboard to manage submissions, mark them as seen/reviewed, and maintain transparency.  
The app promotes respectful communication, faster issue resolution, and a better dining experience for everyone.
